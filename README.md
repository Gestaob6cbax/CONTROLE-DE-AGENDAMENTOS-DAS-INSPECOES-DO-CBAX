# 📋 Portal de Agendamento de Inspeções de Saúde 2026 - CBA X

Este repositório é a **porta de entrada central** para o sistema de agendamento online das inspeções de saúde do **Comando de Bombeiros de Área (CBA X)**.

## 🚀 Objetivo do Projeto
O objetivo deste portal é facilitar o acesso dos militares aos agendamentos de diferentes especialidades, utilizando uma interface visual baseada em ícones (desenhos) para tornar a navegação intuitiva e rápida.

## 🛠️ Como o Sistema Está Estruturado
O portal redireciona o usuário para três sistemas de agendamento independentes, cada um com sua própria base de dados e planilha de controle:

1.  **🩺 Inspeção Médica:** Foco em clínica geral e revisões anuais.
2.  **🦷 Inspeção Odontológica:** Foco em saúde bucal e prevenção.
3.  **🔍 Inspeção Dermatológica:** Foco em avaliações de pele.

## ⚙️ Tecnologias Utilizadas
Para garantir a gratuidade e a eficiência do serviço para o B6, utilizamos:
* **GitHub Pages:** Para hospedagem dos formulários e deste portal.
* **Firebase:** Para garantir que cada militar (por RG) faça apenas um agendamento por especialidade.
* **Google Sheets API:** Para integração direta com as planilhas administrativas da unidade.

## 👥 Administração
Este projeto é gerenciado pela **Sexta Seção (B6) do CBA X**, focado na automação de processos administrativos e gestão de escalas de serviço.
